# MakeSyLifeServer
SyLifeのセットアップ用スクリプト


# Env
- Ubuntu Server 22.04
- [sylife.jp](https://sylife.jp)

# Install
- [OpenSiv3D](https://github.com/Siv3D/OpenSiv3D) v6.5.0
- [Nginx](https://nginx.org/)
- [NodeJS](https://nodejs.org/)
- [Emscripten](https://emscripten.org/)
- gcc 9 (not 11 because of OpenSiv3D version)

# Usage
`bash setup.sh`
or
`./setup.sh`


not use
`sh setup.sh`
